# CAFG Game Modes & Scenarios

## Campaign Mode

### Campaign Structure

**Single Campaign: ~20 missions**
```
Act 1: Introduction (5 missions)
  - Learn mechanics
  - Build starting fleet
  - Defeat first enemy

Act 2: Expansion (7 missions)
  - Conquer territory
  - Unlock new tech
  - Defeat powerful enemies

Act 3: Endgame (5 missions)
  - Final confrontation
  - Ultimate weapon threat
  - Climactic battle

Secret Missions (3+)
  - Hidden bosses
  - Challenge battles
  - Optional extras
```

### Campaign Features

**Persistence**
```
Fleet carries between missions:
  - Ships keep damage
  - Resources accumulate
  - Tech unlocks permanent
  - Upgrades carry over

Campaign World:
  - Faction relationships
  - Territory control
  - Resource nodes
  - Enemy positions
```

**Difficulty Scaling**
```
Easy: Enemies weak, resources abundant
Normal: Balanced experience
Hard: Challenging, limited resources
Insane: Extreme difficulty, 1 life

Scaling per mission:
  Based on fleet power
  Based on player performance
  Based on difficulty selected
  Dynamic adjustment
```

**Story Integration** (Optional)
```
Narrative Arc:
  - Briefing before mission
  - Dialogue during combat
  - Debriefing after mission
  - Character development
  - Plot twists

Voice Acting: Minimal (UI text)
Cinemaics: Optional short clips
Story Themes: Conflict, expansion, survival
```

## Mission Types

### 1. Assault Mission

**Objective**: Destroy enemy space station

```
Environment:
  - Open space with asteroids
  - Enemy fleet defending
  - Station in center
  - Escape point at edge

Win Condition:
  - Station destroyed
  - Time limit: 30 minutes
  - Save mothership

Lose Condition:
  - Mothership destroyed
  - Time limit exceeded
  - Lose 80% of fleet

Rewards:
  - 5000 minerals
  - 500 tech points
  - Unlock new tech
  - Territory control

Difficulty Modifiers:
  Easy: Station weak, few defenders
  Normal: Balanced
  Hard: Strong defenders, reinforcements
  Insane: Multiple stations, endless waves
```

### 2. Defense Mission

**Objective**: Protect friendly station from attackers

```
Environment:
  - Your station in center
  - Asteroids around
  - Enemies attack from edges
  - Escape pods available

Win Condition:
  - Survive 20 minutes
  - Station hull > 50%
  - Enemy fleet destroyed

Lose Condition:
  - Station destroyed
  - Time exceeded
  - Lost 90% of defending fleet

Mechanics:
  - Station auto-defends
  - Can repair station (limited)
  - Reinforcements arrive periodically
  - Enemies escalate over time

Rewards:
  - 3000 minerals
  - 300 tech points
  - Station upgrades
  - Resources from defeated enemies
```

### 3. Fleet Battle

**Objective**: Defeat enemy fleet in open combat

```
Environment:
  - Asteroid field
  - Both fleets start opposite corners
  - No base objectives
  - Pure combat arena

Win Condition:
  - Destroy all enemy ships
  - Kill enemy flagship
  - Time limit: 25 minutes
  - Achieve kill ratio 3:1

Lose Condition:
  - Mothership destroyed
  - Lose 80% of fleet
  - Time exceeded
  - Flee mission

Variants:
  Equal Forces: Mirror-matched fleets
  Underdog: You outnumbered 2:1
  Swarm: Many weak vs few strong
  Sandwich: Enemy approaches from 2 sides

Rewards:
  - 2000 minerals
  - 200 tech points
  - Captured tech from enemies
  - Ship salvage
```

### 4. Convoy Escort

**Objective**: Protect transport ships reaching destination

```
Environment:
  - Open space
  - Transport ships move toward goal
  - Enemy raiders attack
  - Multiple ambush points

Mechanics:
  - Transports are slow
  - Take damage, can be destroyed
  - You escort (no rush)
  - Lose transport = fail condition

Win Condition:
  - 4/5 transports survive
  - Reach destination
  - Defeat all interceptors

Lose Condition:
  - Lost 3+ transports
  - Mothership destroyed
  - Time exceeded (50 minutes)

Rewards:
  - Resources from transports
  - 1000 minerals
  - 100 tech points
  - Technology if escorts win decisively

Twists:
  - Surprise reinforcements
  - Transport breakdown
  - False destination
  - Traitor in convoy
```

### 5. Boss Battle

**Objective**: Defeat experimental superweapon ship

```
Environment:
  - Large arena
  - Boss in center
  - Minion spawning areas
  - Debris obstacles

Boss Mechanics:
  Phase 1: Normal combat
    - Massive health pool
    - Strong weapons
    - Spawns minions

  Phase 2 (50% HP): Enraged
    - 50% faster attacks
    - More minions
    - Special ability charged

  Phase 3 (25% HP): Desperate
    - Uses superweapon
    - All-or-nothing attacks
    - Final wave minions

Special Abilities:
  - Massive laser beam
  - Shockwave attack
  - Teleport
  - Shield overload
  - EMP pulse

Win Condition:
  - Boss destroyed
  - Time: No limit (endurance)
  - Must not lose mothership

Lose Condition:
  - Mothership destroyed
  - Lose 90% of fleet
  - Time: Mission doesn't time out

Rewards: (VERY HIGH)
  - 10000 minerals
  - 1000 tech points
  - Unique weapon unlock
  - New ship blueprint
  - Story progression

Difficulty:
  Easy: Boss weaker, fewer minions
  Normal: Standard
  Hard: Stronger, more minions
  Insane: 2-3 bosses attacking
```

### 6. Survival Mode

**Objective**: Survive endless enemy waves

```
Environment:
  - Arena with asteroid obstacles
  - You start in center
  - Enemies spawn at edges
  - Resources float after kills

Mechanics:
  Wave 1-5: Easy fighters
  Wave 6-10: Mixed cruisers
  Wave 11-15: Capital ships
  Wave 16+: All enemy types

  Resources: Can repair/rebuild
  Reinforcements: Can't call in (limited)
  Time: Infinite until defeat

Win Condition:
  - No win condition (endless)
  - Personal best survival time
  - High score based on kills

Lose Condition:
  - Mothership destroyed
  - Lose all ships
  - All enemies too strong

Rewards:
  - Scaling based on waves survived
  - 100 minerals per wave
  - 10 tech points per wave
  - High score tracking
  - Achievements

Leaderboard:
  - Longest survival time
  - Most kills
  - Highest damage
  - Lowest damage taken
```

### 7. Territory Control

**Objective**: Capture and hold map control points

```
Environment:
  - Large arena
  - 3 control points
  - Asteroid clusters
  - Neutral territories

Mechanics:
  - Control points grant resources
  - Control = 100 minerals/min
  - Ships in area capture point
  - Enemy presence prevents capture
  - Dynamic ownership changes

Win Condition:
  - Control all 3 for 2 min
  - Control 2/3 for 10 min
  - Destroy enemy flagship

Lose Condition:
  - Lose all ships
  - Lose control for 10 min
  - Mothership destroyed

Rewards:
  - Resources earned during mission
  - 1000 minerals bonus
  - 200 tech points
  - Territory ownership

Variants:
  2v2 Team Battle
  3v3v3 Chaos
  1v1v1 Free-for-all
```

### 8. Mining Expedition

**Objective**: Harvest resources from dangerous asteroid field

```
Environment:
  - Dense asteroid field
  - Few enemies (pirates)
  - Resources scattered
  - Limited visibility

Mechanics:
  - Mining ships collect asteroids
  - Pirates attack (random spawn)
  - Protect miners
  - Return to base

Win Condition:
  - Collect 5000 minerals
  - All miners return home
  - Defeat all pirates

Lose Condition:
  - Lost all miners
  - Mothership destroyed
  - Time exceeded (30 min)

Rewards:
  - All collected resources
  - Bonus for protecting miners
  - 500 minerals
  - 50 tech points

Difficulty:
  Easy: Lots of resources, few pirates
  Hard: Scarce resources, many pirates
```

## Skirmish Mode

**Quick battles without campaign progression**

```
Features:
  - Customizable fleet
  - Choose enemy fleet
  - Set map/asteroids
  - Set difficulty
  - No progression (practice)

Rules:
  - No time limit
  - First to destroy other fleet wins
  - Can surrender
  - Can retry

Uses:
  - Test new ship designs
  - Practice strategies
  - Fun casual play
  - Warm-up before campaign
```

## Scenario Mode

**Pre-made balanced scenarios**

```
Scenario Types:
  Classic: Balanced 1v1
  Asymmetric: Uneven forces
  Chaos: Multi-faction
  Challenge: Special rules
  Custom: Player-made

Examples:
  "David vs Goliath": 5 fighters vs 1 battleship
  "Final Stand": Defend station vs army
  "Blitz War": 5-minute time limit
  "Economic Warfare": Resource gathering focus
```

## Game Settings & Modifiers

### Difficulty Modifiers

```
Gameplay Difficulty:
  Easy: +50% player HP, -50% enemy damage
  Normal: Balanced
  Hard: -25% player HP, +25% enemy damage
  Insane: -50% player HP, +50% enemy damage

Resource Difficulty:
  Abundant: +100% resource income
  Normal: Standard
  Scarcity: -50% resources
  Hardcore: Very limited resources

Enemy Difficulty:
  Dumb: Enemies make mistakes
  Normal: Balanced AI
  Smart: Intelligent enemy strategy
  Genius: Perfect enemy tactics

Time Limit Difficulty:
  Relaxed: No time limits
  Normal: Standard time limits
  Rush: 50% less time
  Blitz: 75% less time
```

### Custom Rules

```
Optional Rules:
  - No shields (pure armor)
  - No capital ships (small units only)
  - Friendly fire on/off
  - Fast-forward speed available
  - Debug mode (see all)
  - God mode (unbeatable)
```
